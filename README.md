# ivinay-website

Static personal website for Vinay Kumar Polamarasetty.

## Files
- `index.html` — complete website
- `vercel.json` — optional clean Vercel configuration

## Deploy to Vercel
1. Upload the files in this folder to the root of your GitHub repository.
2. In Vercel, choose **Add New → Project**.
3. Import the `ivinay-website` GitHub repository.
4. Leave the framework preset as **Other**.
5. Click **Deploy**.
6. In Vercel, open **Project → Settings → Domains**.
7. Add `ivinay.com` and follow the DNS instructions shown by Vercel.
